FSL is a comprehensive library of image analysis and statistical tools for FMRI, MRI and DTI brain imaging data. FSL is written mainly by members of the Analysis Group, FMRIB, Oxford, UK.

Software
FSL runs on Apple, PCs (Linux and Windows) and Sun, and is very easy to install. Most of the tools can be run both from the command line and as GUIs ("point-and-click" graphical user interfaces). See FSL screenshots and videos. There is also an evaluation and example data package available to accompany FSL.

Support
For up-to-date information regarding FSL see the central version of this FSL home page. Commonly-asked questions are answered in the FSL FAQ. We run a course on FSL every year or so - view or download the lectures and course practical data. For further support relating to FSL or related theory, join the FSL email list.
